## ✨ Overview

Tic-Tac-Toe Premium is a modern reimagining of the classic game with a futuristic neon aesthetic and polished user experience.

The project includes:

* Single Player vs AI Mode
* Local Two Player Mode
* Minimax AI Algorithm
* Particle Celebration Effects
* Dynamic Sound Generation
* Responsive Mobile Support
* Live Score Tracking
* Glassmorphism Inspired Interface

All functionality is contained within a single **index.html** file.

---

## 🎮 Features

### Game Modes

* 👥 Local Two Player Mode
* 🤖 Player vs Computer Mode

### Smart AI Opponent

The computer opponent uses the **Minimax Algorithm**, making it extremely difficult to defeat.

Features:

* Optimal move calculation
* Instant board evaluation
* Strategic decision making
* Unbeatable gameplay logic

### Premium User Interface

* Neon visual theme
* Animated glowing effects
* Glassmorphism styling
* Smooth transitions
* Responsive layout
* Interactive buttons

### Audio System

Built using the Web Audio API.

Includes:

* Move sound effects
* Victory sounds
* Defeat sounds
* Draw notifications
* Dynamic tone generation

No external audio files required.

### Particle Effects

Winning and game-ending moments trigger animated particle explosions for enhanced visual feedback.

### Score Tracking

Track:

* Player Wins
* Computer Wins
* Draws

Scores can be reset at any time.
